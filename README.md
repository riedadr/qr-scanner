# QR Scanner

made with [mebjas/html5-qrcode](https://github.com/mebjas/html5-qrcode) in NextJS.

## Getting started

### Development

```sh
git clone https://github.com/riedadr/qr-scanner.git
```

```sh
npm install
```

```sh
npm run dev
```

### Production

```sh
npm install
```

```sh
npm run build
```

```sh
npm run start
```
